# Benefit Bistro

## Getting started

1. create a Python virtual environment
2. install dependencies
3. run the application

### For Linux and Mac

```bash
python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python src/main.py
```

### For Windows

```bash
py -3.11 -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python src/main.py
```
